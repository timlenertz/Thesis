Detailed Description

template<typename Cloud_fixed, typename Cloud_loose>
class pcf::same_point_correspondences< Cloud_fixed, Cloud_loose >

Point correspondences between points at same indexed in both clouds.

Both clouds need to have same size. Assumes one cloud is a transformation of the other. Used for measuring registration error with known correspondences.
